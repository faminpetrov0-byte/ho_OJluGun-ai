**Локация:** `/src/core/execution/ExecutionPlanManager.ts`

**Революционные возможности:**

- **Dependency management** - умные зависимости между шагами

**Интеграция с Cline:**

### ⚡ **2. MAX_CORRECTION_DEPTH в ToolExecutor.ts** - УМНАЯ КОРРЕКЦИЯ
**Локация:** `/src/core/task/ToolExecutor.ts`

**Прорывная технология:**

- **AI-анализ ошибок:** Автоматическое определение причин сбоев
- **Safety checkpoints:** Git stash перед рискованными операциями
- **Аварийный откат:** Восстановление состояния при критических ошибках
- **Статистика попыток:** Отслеживание эффективности коррекций

**Интеграция с Cline:**
- Встроен в основной цикл выполнения инструментов
- Использует существующий `executeCommandTool` для git операций
- Совместим с системой `say/ask` для уведомлений пользователя
- Интегрирован с `coordinator` для управления инструментами
